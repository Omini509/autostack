# Setup

Clone the repo.
```
git clone https://github.com/elijahsawyers/AutoStack.git
```

Navigate to the project directory, and install the requirements.
```
pip install -r requirements.txt
```

Run the install bash script.
```
./install.sh 
```

Start listening for errors!
```
autostack
```
